# Weather app

A sillz little weather app, designed to fetch weather from `visualcrossing.com` and get a corresponding gif from `giphy.com`.

Built only using vanilla HTML, CSS and JavaScript.

## Prerequisites

- Node.js v18 or later
- npm or yarn

## Installation

```bash
# Clone the repository
git clone git@github.com:goran1010/weather-api-test.git

# Navigate to project directory
cd weather-api-test

# Install dependencies
npm install
```

## Usage

```bash
# Start development server with live reload
npm run dev

# Build for production
npm run build

# Run tests
npm test

# Format code
npm run format

# Lint code
npm run lint

# Deploy to GitHub Pages
npm run push-gh-pages
```
